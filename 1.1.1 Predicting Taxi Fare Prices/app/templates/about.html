About This Project

This web app demonstrates the NYC Taxi Fare Prediction project. It estimates the cost of a taxi ride in New York City based on pickup and drop-off locations, travel distance, and time-related information. The goal is to provide accurate and interactive fare predictions for educational and demonstration purposes.

How it Works

  • Data preprocessing and feature engineering for the taxi trip dataset.
  • Model training with a regression-based approach to predict fares.
  • Feature extraction includes distance, hour of day, day of week, and month.
  • FastAPI backend serving predictions via a REST API.
  • Interactive front-end allows both manual input and batch file uploads.
  • Real-time feedback displays estimated fare instantly below the input form.

Why This Project is Useful

Understanding taxi fare dynamics can help:

  • Students and developers learn machine learning workflow end-to-end.
  • Experiment with feature engineering and model deployment.
  • Demonstrate practical applications of regression modeling in urban transportation.

Disclaimer

This application is intended for educational and demonstration purposes only. Do not use it for real-world fare calculations or business decisions without proper validation.
